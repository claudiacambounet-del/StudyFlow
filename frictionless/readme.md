# Study Flow - Personal Study Assistant!

StudyFlow is a program designed to help students organize their academic life, plan study sessions intelligently, and build better daily routines.

## Features

### Task Management
- Add tasks with estimated time and priority
- Mark tasks as done
- List all tasks
- Delete completed tasks
- Clear all tasks
- Categorize tasks (according to subjects)

### Planning
- Generate a study plan based on available time
- Automatic daily organization

### Routines
- Study routines are integrated
- Reminder system that motivates you to study

### Pomodoro Timer
- 25 minute focus sessions with 5 minute breaks
- Helps boost concentration and productivity
